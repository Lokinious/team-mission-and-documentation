# Team Mission and Documentation

This repository is the shared home for mission, values, team direction, and foundational documentation for Caleb's Startup.

## Purpose
- Align the team around what we are building and why
- Capture lightweight guidance for how we work together
- Give new teammates a simple starting point for understanding the project

## Initial documents
- `docs/mission-and-values.md`
- `docs/working-agreements.md`

## Scope
This repository is intentionally documentation-first. It is not for application code, infrastructure code, or product assets.

## Near-term goals
1. Clarify the startup mission and priorities
2. Define team expectations and collaboration habits
3. Expand shared documentation as the project grows
